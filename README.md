# whiteBoard

A lightweight, collaborative whiteboard app for brainstorming, sketching ideas, and sharing notes in real time.

## Features
- Freehand drawing (pen/eraser)
- Shapes and lines 
- Text notes
- Color + stroke size controls
- Undo / redo + clear canvas
- Export board as PNG
- Shareable board link (optional / if supported)
